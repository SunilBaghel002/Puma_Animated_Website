# Velocity X Images

This directory should contain 120 webp images named `1.webp` through `120.webp` showing a 360° rotation of the Puma Velocity X shoe.

## Image Requirements
- Format: WebP
- Naming: 1.webp, 2.webp, ..., 120.webp
- Content: Sequential frames of the shoe rotating 360°
- Recommended resolution: 800x800 or higher

## How to Generate
You can create these images using:
1. 3D rendering software (Blender, Cinema 4D, etc.)
2. Product photography turntable
3. AI image generation tools
